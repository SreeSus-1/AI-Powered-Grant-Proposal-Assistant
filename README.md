# AI-Powered-Grant-Proposal-Assistant
Assist researchers or nonprofits in drafting grant proposals.

This project is a multi-agent grant proposal drafting assistant designed to help researchers and nonprofits prepare funding applications more efficiently. It uses an Outline Designer Agent to create the structure of the proposal, a Budget Estimator Agent to generate a draft funding breakdown, and a Reviewer Simulation Agent to critique the generated proposal with realistic feedback. These agents work together to turn a topic, set of goals, and funding agency into a more complete proposal draft. The system also stores proposal versions and rationale in memory so users can revisit earlier drafts and track how the proposal evolves over time.
